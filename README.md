## Budgeting App

To help those in need of financial structure and sorting in daily lives (like me :) ).

### API

https://api.ynab.com/ - **YNAB**

- You'll likely need to create a account, generate a token (in this repo I left my open for now, after lecturer's review I'll make it secret).
- There is a request cap at 200 per hour. 
- Here is Endpoints list if interested - **https://api.ynab.com/#endpoints**

This is just a personal/educational project, I'll update it whenever I get a chance, or I'll finish it for final project, but let's hope for the best! :)

For other running steps:
- app doesn't require login, just make sure to provide your account token
- most of cards have static data, even PDF parser if fake... (for now!)
- when adding items in "Future expanses" section make sure to not make spelling errors and set date more than current (e.g. current 21 July, set it to 23 July)
